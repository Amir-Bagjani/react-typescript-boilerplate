# React typescript boilerplate

To run this project

```bash
npx degit https://github.com/Amir-Bagjani/react-typescript-boilerplate project
```

or clone repository

```bash
  git clone https://github.com/Amir-Bagjani/react-typescript-boilerplate.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Build

```bash
  npm run build
```
